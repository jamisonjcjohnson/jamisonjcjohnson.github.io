This guide explains how to install AnyDesk on Windows and macOS so remote assistance can be available when needed. These steps cover the normal installation and basic setup only. Optional unattended access can be configured later if necessary and agreed upon.

## What you will need

- A Windows PC or Mac with internet access
- Permission to install software on the computer
- The official [AnyDesk installer](https://anydesk.com/en/downloads/)


## Windows Installation Instructions

### 1. Download AnyDesk

1. Open your web browser.
2. Go to the official [AnyDesk installer](https://anydesk.com/en/downloads/) page.
3. Download the Windows installer.

### 2. Install AnyDesk

1. Open the downloaded AnyDesk file.
2. If prompted, choose to install AnyDesk on the device.
3. Accept any Windows security or administrator prompts.
4. Complete the installation.
5. Open AnyDesk after the install finishes.

### 3. Confirm the app opens correctly

1. When AnyDesk opens, locate the device's AnyDesk ID or alias in the main window.
2. Keep the application open when you are ready for support.
3. Share the AnyDesk ID with technical support when they are going to connect.

### 4. During a normal support session

1. The AnyDesk ID will be entered from the technical POC side.
2. You will receive a connection request.
3. Click **Accept** when you are ready for the connection.
4. Stay near the computer in case permissions or prompts appear during the session.

## macOS Installation Instructions

### 1. Download AnyDesk

1. Open your web browser.
2. Go to the official [AnyDesk installer](https://anydesk.com/en/downloads/) page.
3. Download the macOS version of AnyDesk.

### 2. Install AnyDesk

1. Open the downloaded `.dmg` file, which is likely in the `Downloads` directory.
2. In the installer window, open the AnyDesk application.
3. Click **Install Now** if prompted.
4. Accept any prompts needed to complete the installation.
5. Move AnyDesk to the **Applications** folder if requested.
6. Launch AnyDesk.

### 3. Grant required macOS permissions

macOS may require additional permissions before remote control works properly.

1. Open AnyDesk.
2. Follow any prompts shown by the app.
3. Allow the requested permissions in **Privacy & Security** settings.
4. Approve permissions such as:
   - Screen Recording
   - Accessibility
   - Full Disk Access
5. Restart AnyDesk if macOS requires it after changing permissions.

### 4. Confirm the app opens correctly

1. In the main AnyDesk window, locate the device's AnyDesk ID or alias.
2. Keep the application open when you are ready for support.
3. Share the AnyDesk ID with technical support when they are going to connect.

### 5. During a normal support session

1. The AnyDesk ID will be entered from the technical POC side.
2. You will receive a connection request.
3. Click **Accept** when you are ready for me to connect.
4. Stay near the Mac in case additional permission prompts appear.

## General Notes

- Download AnyDesk only from the official [AnyDesk installer](https://anydesk.com/en/downloads/) page.
- Keep the app updated if you use it regularly.
- If the app opens but remote control does not work on a Mac, the missing step is usually one of the macOS permissions.
- If you close the app completely, technical support may not be able to connect until you reopen it.

## Optional Note: Unattended Access

Unattended access is not required for a normal support session. It should only be configured if needed and if you agree to it.

If we decide to enable it later, the general process is:

1. Open AnyDesk.
2. Go to **Settings > Access**.
3. In the **Unattended Access** section, click **Set password**.
4. Create a strong password and save it.

On macOS, the required system permissions must also already be granted for remote control to work properly.

For more detailed and official documentation, refer to the [AnyDesk Help Center](https://support.anydesk.com/docs/getting-started/).
